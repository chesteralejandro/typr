Typewriter Effect Library

A lightweight, dependency-free JavaScript library for creating smooth, customizable typewriter animations using simple data-* attributes.

Quick Start

1️⃣ Add the Target Element

<span class="typewriter1"></span>

2️⃣ Include the Script

<script
       defer
       src="https://chesteralejandro.github.io/poc-typewriter-effect/typewriter.js"
       data-class="typewriter1"
       data-word-list='["Web Developer.", "Designer.", "Coder."]'
></script>
⚠️ IMPORTANT: The element's class value MUST be exactly the same as the script's data-class value.

Attribute Explanation

Attribute Description
src URL of the library file.
defer Ensures the script runs after HTML parsing.
data-class The class of the element to animate. Must match the element’s class value exactly.
data-word-list JSON array of phrases to type.
